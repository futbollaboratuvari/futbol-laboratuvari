Futbol Laboratuvari Site Son Surum
=================================

Bu paket GitHub Pages icin hazirlanan guncel Futbol Laboratuvari web sitesi dosyalarini icerir.

Degisen ana dosyalar:
- index.html: Kurucu bolumu, Medya Galerisi, Spor Toto Performansi, Analiz Veritabani, Basari Arsivi baglantilari ve bolumleri guncellendi.
- style.css: Siyah/dore/neon yesil premium tasarim, kurucu karti, medya galerisi, Spor Toto kartlari ve mobil uyum stilleri guncellendi.
- script.js: 6 genel analiz kaydi, 15 Spor Toto kaydi, ana panel ozet hesaplari, analiz kartlari, Spor Toto kartlari ve arsiv tablolarinin veri uretimi eklendi.

Eklenen/kullanilan gorseller:
- assets/images/fl-logo-mark.webp: Ust menu ve hero logo kullanimi.
- assets/images/fl-logo-gallery.webp: Medya galerisi logo karti.
- assets/images/hero-studio.webp: Hero arka plan gorseli.
- assets/images/cem-kaplanoglu-founder.webp: Cem Kaplanoglu kurucu karti gorseli.
- assets/images/gallery-analiz-serisi.webp: Instagram / analiz serisi gorseli.
- assets/images/gallery-spor-toto.webp: Spor Toto dosyasi gorseli.
- assets/images/gallery-yorum-kosesi.webp: Yorum Kosesi gorseli.
- assets/images/gallery-marka.webp: Futbol Laboratuvari marka gorseli.

Canli siteye yukleme adimlari:
1. GitHub'da futbollaboratuvari/futbol-laboratuvari reposunu acin.
2. Paket icindeki index.html, style.css ve script.js dosyalarini repo kok dizinine yukleyin veya mevcut dosyalarla degistirin.
3. Repo kok dizininde assets/images/ klasoru yoksa olusturun.
4. Paket icindeki assets/images/ klasorundeki tum .webp dosyalarini repo icindeki assets/images/ klasorune yukleyin.
5. Degisiklikleri commit edin. Ornek commit mesaji: "Futbol Laboratuvari PRO 7.0 site guncellemesi".
6. GitHub Pages yayini main branch'ten calisiyorsa commit sonrasi site otomatik guncellenir.
7. Canli siteyi kontrol edin: https://futbollaboratuvari.github.io/futbol-laboratuvari/

Notlar:
- Dosya yollari gorelidir; GitHub Pages kok dizin yapisini bozmayin.
- index.html icindeki gorsel yollarinin assets/images/... olarak kalmasi gerekir.
- Kurucu karti bagimsiz bolum olarak duzenlenmistir; hero yazisi kurucu fotografi uzerine tasmaz.
